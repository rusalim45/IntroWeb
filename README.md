## Overview

This repository contains all laboratory assignments completed for the Intro to Web Programming course. The purpose of this repository is to demonstrate practical skills in web development, starting from basic HTML structure and progressing toward dynamic, interactive, and data-driven web applications.

Each lab reflects step-by-step improvement in understanding core web technologies and applying them in real-world scenarios. The work shows consistent development practices, organized file structure, and continuous enhancement of coding skills.

## Learning Objectives

- Build structured and semantic web pages using HTML.
- Apply CSS styling techniques, including responsive layouts with Flexbox and Grid.
- Implement interactivity using JavaScript.
- Develop basic server-side functionality using technologies such as Python (Django/Flask), Node.js, or PHP.
- Create dynamic front-end interfaces using modern frameworks like React, Angular, or Vue.
- Design and integrate RESTful APIs for client-server communication.
- Apply web accessibility standards and responsive design principles.
- Implement basic web security practices such as input validation, XSS protection, and CSRF prevention.
- Use Git for version control and structured project management.
